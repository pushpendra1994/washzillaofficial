# WashZilla Website

Premium static landing page for WashZilla, a car and bike washing center near Tia Hospital, IIIT Chauraha, Jalwa, Prayagraj.

## Open Locally

Open `index.html` directly in a browser.

## Personalize

- WhatsApp number is set in `script.js`:
  `const phoneNumber = "917869697297";`
- Display contact numbers: `7398233605 / 7869697297`.
- The attached WashZilla logo is stored at `assets/washzilla-logo.png`.
- The gallery uses generated starter vehicle visuals. Replace them with real before/after customer vehicle photos after launch.

## Files

- `index.html` - page structure and content
- `styles.css` - premium black and gold responsive styling
- `script.js` - mobile menu and WhatsApp booking flow
- `assets/washzilla-hero.png` - generated hero visual
- `assets/washzilla-logo.png` - WashZilla logo
- `assets/gallery-*.png` - moving before/after wash gallery visuals
